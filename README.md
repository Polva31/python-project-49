## Описание
Набор мини-игр для тренировки мозга.

## Установка
```bash
make install
```

## Использование
```bash
brain-games
```

## Игры

### Brain Even
Определение чётного числа

### Brain Calc
Вычисление арифметических выражений

### Brain GCD
Нахождение наибольшего общего делителя

### Brain Progression
Поиск пропущенного числа в прогрессии

### Brain Prime
Определение простого числа
